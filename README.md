# 📘 Curso Java — Fundamentos de POO

Repositório criado para armazenar os códigos, exemplos e práticas desenvolvidas durante o curso **Fundamentos de Programação Orientada a Objetos em Java**, da **KipperDev**.

Este projeto tem como objetivo consolidar os conceitos fundamentais da **Programação Orientada a Objetos (POO)** no ecossistema Java, com foco em organização de código, modelagem de classes e boas práticas de desenvolvimento.

---

## 🧠 Conteúdos abordados

Durante o curso, foram estudados os seguintes tópicos:

- Conceitos fundamentais de Programação Orientada a Objetos  
- Criação e estrutura de **classes e objetos**  
- Uso de **atributos e métodos**  
- Encapsulamento e modificadores de acesso  
- Construtores em Java  
- Organização e responsabilidade de classes  
- Boas práticas de modelagem orientada a objetos  
- Introdução à abstração e reutilização de código
